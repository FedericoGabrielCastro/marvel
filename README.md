<h1 align="center">
    Instructions
</h1>

#### Installing Node Modules

```bash
yarn install
```

#### Starting the Application

```bash
yarn run dev
```

#### Building for Production

```bash
yarn build
```

<h1 align="center">
    For devs
</h1>

### Exectue prettier --write.

```bash
yarn run prettier
```
